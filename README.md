
  # Upgrade Techinnov Website Design

  This is a code bundle for Upgrade Techinnov Website Design. The original project is available at https://www.figma.com/design/IjlDDJDMTFgiIZuW20df6o/Upgrade-Techinnov-Website-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Production Build

  The current setup uses the Tailwind CSS CDN, which is not recommended for production. For a production build, you should install Tailwind CSS as a PostCSS plugin.
  See the official guide: https://tailwindcss.com/docs/installation
